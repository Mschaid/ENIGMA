split_train_by_query(column_name, cutoff, processed_data=False, data=None)#

Summary#

Split the processed data into training and testing sets based on a specified column’s cutoff value.

param column_name:

Name of the column to split the data on.

type column_name:

str

param cutoff:

The cutoff value to use to split the data. Any value less than or equal to the cutoff will be in the training set, while any value greater than the cutoff will be in the testing set.

type cutoff:

float

returns:

  • None

  • Notes

  • —— – Updates the class instance with four attributes - X_train, X_test, y_train, and y_test. X_train and y_train are the training features and target data, respectively, which consist of rows from the processed data where the value of the specified column is less than or equal to the cutoff. X_test and y_test are the testing features and target data, respectively, which consist of rows from the processed data where the value of the specified column is greater than the cutoff.
